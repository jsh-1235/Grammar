Object

Object 객체는 객체의 가장 기본적인 형태를 가지고 있다.

다시 말해서 아무것도 상속받지 않은 순수한 객체다.

모든 객체의 부모 객체이다.

hasOwnPropery는 인자로 전달된 속성의 이름이 객체의 속성인지 여부를 판단한다.

Data type

객체가 아닌 데이터 타입을 원시 데이터 타입(Primitive type)이라고 한다.

Number, String, Boolean, null, undefined

래퍼 객체 (Wrapper Object)

래퍼 객체로는 String, Number, Boolean이 있다. null과 undefined는 존재하지 않는다.

참조 vs 복제

원본 파일에 대한 심볼릭 링크(Symbolic Link)를 통해서 저장된 원본의 주소를 참조해서 원본의 위치를 알아내고 원본에 대한 작업을 하게 된다.
